IPL Data Analysis (2008–2023)
Project Overview
This project performs Exploratory Data Analysis (EDA) on Indian Premier League (IPL) match data from 2008 to 2023.

The objective is to uncover meaningful insights about:
Team performance
Batting dominance
Match-winning trends
Toss decision impact
Overall historical patterns
The analysis focuses on identifying statistical trends and visualizing relationships within the dataset.

Dataset Description
The dataset contains:
1️) Match Data
Ball-by-ball records
Runs scored
Batting and bowling teams
Player information
2️) Match Information Data
Match winner
Toss winner
Toss decision
Venue details
Match outcomes
The dataset covers IPL seasons from 2008 to 2023.

Exploratory Data Analysis (EDA)
Exploratory Data Analysis involves:
Summarizing main characteristics of data
Identifying trends and patterns
Detecting relationships between variables
Visualizing insights through graphs
EDA does not build predictive models; instead, it helps understand the dataset deeply.

Visualization Techniques Used
Bar plots
Aggregation using groupby
Data summarization
Comparative performance analysis
Libraries used for visualization:
Matplotlib
Seaborn

Technologies Used
